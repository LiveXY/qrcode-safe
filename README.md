# 安全二维码保险箱 (SecureQR Vault)

这是一个专注于安全性的二维码管理应用，专为 macOS 电脑端和 iPhone 手机端优化设计。该应用允许用户扫描二维码，使用 AES-256 标准对数据进行本地加密，并以文件的形式安全保存。

## 💡 设计提示词与需求 (Project Requirements)

本项目基于以下核心需求开发：

1.  **多端支持**：应用需要同时支持 MAC 电脑端和 iPhone 手机端，界面需具备良好的响应式设计。
2.  **安全至上**：
    *   使用 AES-256 高级加密标准。
    *   所有数据处理（扫描、加密、解密）均在本地浏览器环境完成，不经过任何服务器，确保数据隐私。

## 📖 使用说明 (User Manual)

### 1. 启动应用
*   应用采用 React 开发，响应式布局。
*   在手机端访问时，会自动适配全屏模式。

### 2. 扫描与加密 (Scan & Encrypt)
1.  点击主界面的 **“扫描二维码”** 按钮。
2.  授予相机权限，将摄像头对准目标二维码。
3.  扫描成功后，系统将展示解码后的明文内容。
4.  点击 **“保存加密数据 (.qrd)”** 按钮：
    *   **`.qrd` 文件**：包含 AES-256 加密后的数据（JSON 格式）。
    *   **`.env` 文件**：包含用于解密的密钥 (KEY) 和偏移量 (IV)。
    *   *⚠️ 重要提示：请务必妥善保管 `.env` 密钥文件，丢失将导致数据无法找回。*

### 3. 读取与解密 (Read & Decrypt)
1.  在主界面点击 **“读取本地文件”**。
2.  从设备中选择之前保存的 `.qrd` 文件。
3.  系统将自动尝试使用当前的加密上下文进行解密。
    *   *注意：在当前演示版本中，解密依赖于当前会话生成的密钥。在实际生产环境中，通常需要手动导入对应的 `.env` 密钥文件。*
4.  解密成功后，屏幕将显示原始明文数据。

### 4. 生成二维码 (Generate QR)
1.  在解密结果页面，点击 **“生成二维码”** 按钮。
2.  屏幕中央将弹出一个高清晰度的二维码，包含了当前的解密数据，可供其他设备再次扫描。

## 🛠 技术栈 (Tech Stack)

*   **核心框架**: React 18 + TypeScript
*   **UI 样式**: Tailwind CSS (支持 Dark Mode 和移动端适配)
*   **加密算法**: Crypto-js (AES-256, CBC 模式, PKCS7 填充)
*   **二维码能力**: 
    *   扫描: `jsQR`
    *   生成: `qrcode.react`
*   **图标**: Lucide React

## 🔒 安全性细节

*   **AES-256**: 采用 256 位密钥长度的对称加密，目前被认为是量子计算机出现前最安全的加密方式之一。
*   **随机 IV**: 每次会话生成随机初始化向量 (IV)，确保相同的明文加密后产生不同的密文，防止彩虹表攻击。
*   **离线可用**: 应用加载后无需网络连接即可完整运行。
